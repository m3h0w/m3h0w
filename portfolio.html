Selected Projects

Endoscopy Quality Evaluation

Developed software created a real time map for assesing quality of a colonoscopy exam. The project was developed as a POC desktop app based on gastroentherologist's expertise. I was focused on optical flow map that I used to calculate the distance travelled by the endoscopy camera inside patient's body.

--- C++, OpenCV, OpticalFlow ---

Parking Occupancy Detection

Real-time detection of parking space occupancy. The project was a POC that aimed to replace expensive sensors. I used traditional computer vision techniques.

--- Python, OpenCV ---

TRAiVEL - Sentiment-based Travel Recommendations

An app that uses sentiment analysis to estimate current mood in European countries based on twitter data and recommended travel destinations together with prices fetched from Skyscanner api and a follow-through links to actual flight tickets for the upcoming weekend. Built in a team of 4, in approx. 24 hours for a hackathon.

--- Python, JavaScript, Flask, Machine Learning ---

Parametric t-SNE with Emotion Recognition

A novel technique for visualizing emotion recognition results including the temporal dimension. An "emotion space" is created using parametric t-SNE, where machine learning models can be visually compared with respect to modalities, models, and hyperparameters used. As a part of the project, we've used KNN and RNNs on the CMU multimodal dataset to perform emotion recognition.

--- Python, Keras, Machine Learning ---

Rule-based Insurance Recommendation - Admin Panel

A versatile web app that allows its user to set rules that get evaluated based on input features. Its primary goal was to create a universal way of deciding which insurance products should be displayed depending on the user. The app communicates with various API's and exposes endpoints for front-ends.

--- Full-stack, Flask, React, SQLite ---

CNNs & Transfer Learning for Furniture Recognition

Project developed for a course, solving Kaggle competition. I used Convolutional Neural Networks to distinguish between 128 classes of furniture. I utilized transfer learning and worked with GPU's in the cloud, on a Linux machine.

--- Azure, CNNs, Keras, Kaggle ---
